Projeto para faculdade!

Farei um programa para visualizar rotas, inspeirado no trabalho de engenharia de requisitos.



RF:
* Criar Rota -- Rotas com 4 endereços dentro do Array
* Editar
* Deletar
* Visualizar


RNF:

* Menu
* Estilização



Sobre o projeto:

Criar Rota:
    Na rota terá a rua e o núemro, terá que ser armazenada dentro de um array e cada array terá a capacidade de 4 endereços somentes.

Editar:
    Pode colocar outra rua ou a mesma caso tenha errado o nome da rua.

Deletar:
    Apaga uma rua e faz com que as outras que estava abaixo suba e libere um espaço no array


Listar:
    Consegue ver a rota, cada rota contém 4 endereços.
